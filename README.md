Vagrant for Docker
---

# Required

* Vagrant
* Virtualbox

# Start 

* Windows
> run.bat


* Mac or Linux
(must create data folder)

> chmod a+x ./run.sh

> mkdir data

> ./run.sh

* Common

> vagrant up --provider virtual box
